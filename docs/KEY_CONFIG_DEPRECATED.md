# SSH 密钥配置说明（已弃用）

本配置文件已弃用。项目现在使用密码认证而不是密钥文件认证。

## 如果您想切换回密钥认证

请参考项目根目录的以下文件：
- `SSH_KEY_SETUP.md` - SSH 密钥配置指南
- `QUICK_START.md` - 快速开始指南

## 当前配置方式

项目现在使用密码认证：
- ✅ 配置简单：只需在 `.env` 中设置 `NAS_PASSWORD`
- ✅ 无需密钥文件管理
- ✅ 适合测试和开发环境

## 密钥认证的优势

如果您想更安全，可以切换回密钥认证：
- ✅ 更安全：私钥不存储在配置文件中
- ✅ 无需输入密码
- ✅ 符合 SSH 最佳实践

详见 `SSH_KEY_SETUP.md` 文件。
